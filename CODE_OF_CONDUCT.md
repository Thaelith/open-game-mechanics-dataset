# Code of Conduct

This project uses the Contributor Covenant Code of Conduct, version 2.1.

Be respectful, constructive, and specific. Disagreements about taxonomy, examples, or design wording should focus on the repository goal: a practical, original, machine-readable dataset of game mechanics.

Unacceptable behavior includes harassment, threats, discriminatory language, personal attacks, and repeated disruption of maintainer decisions.

Report conduct concerns through the security or maintainer contact listed for the project. Maintainers may remove comments, close issues, or block contributors to protect the project community.
