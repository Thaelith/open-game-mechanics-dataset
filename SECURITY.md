# Security Policy

This repository primarily contains JSON data and Python tooling. Please report security issues if you find a vulnerability in the scripts, GitHub Actions workflow, dependency handling, or generated artifacts.

Do not open a public issue for an active security vulnerability. Contact the maintainers privately with:

- affected file or workflow
- reproduction steps
- expected impact
- suggested fix, if available

Dataset content corrections, taxonomy disagreements, and broken mechanic references are not security issues; use the normal issue templates for those.
