# Roadmap

## Near Term

- Review seed entries for wording depth and category placement.
- Add more reviewed examples for underrepresented genres.
- Add category metadata files and validate them with `schema/category.schema.json`.
- Add richer query examples for Python, JavaScript, and command-line users.

## Medium Term

- Add maintainer review states beyond `draft`.
- Add mechanic dependency graphs and visualization exports.
- Add optional implementation pseudocode fields after schema discussion.
- Add localized category and tag labels without changing canonical IDs.

## Long Term

- Support versioned schema migrations.
- Publish package artifacts for npm and PyPI consumers.
- Build a static browser for searching and comparing mechanics.
- Establish maintainer review guidelines for `reviewed` status.
